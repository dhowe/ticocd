## TICOCD

### online
https://rednoise.org/ticocd/

### browser
```
$ git clone https://github.com/dhowe/ticocd.git
$ cd ticocd
then open 'index.html' in your browser (via a local webserver)
(note: use chrome-based browsers for now)
```

### node
```
$ git clone https://github.com/dhowe/ticocd.git
$ cd ticocd
$ npm install
$ node ticocd
```
